# Phitron Batch 9 - Week 2: Practice Day 2 (Module 7.5)

This folder contains C programming solutions for the problems assigned in **Week 2, Practice Day 2, Module 7.5** of the Phitron Batch 9 course.

## Problems Solved

### 1. A - Summation (`A_Summation.c`)
- **Description**: This program reads $N$ integers into an array and calculates the total sum of all elements.
- **Output**: The absolute value of the sum.

### 2. B - Searching (`B_Searching.c`)
- **Description**: This program reads $N$ integers into an array and searches for a specific value $X$.
- **Output**: The index (0-based) of the first occurrence of $X$.

### 3. Reverse and Odd (`Reverse_and_Odd.c`)
- **Description**: This program takes an array of $N$ integers and prints the elements located at **odd indices** in **reverse order**.
- **Logic**: Iterates from the end of the array to the beginning and checks if the index is odd.

### 4. Sum Sum (`Sum_Sum.c`)
- **Description**: This program takes $N$ integers and calculates two separate sums:
    1. Sum of all **positive** numbers.
    2. Sum of all **negative** numbers.
- **Output**: Prints the positive sum and the negative sum separated by a space.

### 5. Update and Print (`Update_and_Print.c`)
- **Description**: This program initializes an array of $N$ integers, then:
    1. Updates an element at a specific index $X$ with a new value $V$.
    2. Prints the entire updated array in **reverse order**.

## How to Run

To compile and run any of these files, use a C compiler (like `gcc`):

```bash
gcc filename.c -o filename
./filename
```
